# Status Note (SillyTavern Extension)

A lightweight Author's Note-like extension:
- Floating window (open/close) occupying the left space between screen edge and chat
- Per-character saved prompt
- In-chat injection at configurable depth and role (System/User/Assistant)
- Optional inclusion in World Info scanning

## Install
Place folder into:
`SillyTavern/data/default-user/extensions/SillyTavern-StatusNote`

Then restart SillyTavern or reload the UI.

## Usage
Open Extensions menu -> **Status Note**.
Set your text, depth & role, optionally allow WI scan, then enable.
